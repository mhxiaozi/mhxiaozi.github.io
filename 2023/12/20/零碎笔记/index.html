零碎笔记

什么是通信

通信就是通过媒介（介质）和行为信息的传递与交流

网络常见设备

  • 交换机：二层（不能配IP地址，是个透明设备）、三层交换机
  • 路由器：处在网络层
  • 防火墙：网络安全设备
  • AC：有三层交换机的一些功能，类似于三层交换机
  • AP

网络常见术语

术语 说明
数据载荷 最终想要传递的数据
报文 网络中交换与传输的数据单元（数据包）
头部 在数据载荷前面添加的信息段
尾部 在数据载荷后面添加的信息段
封装 形成报文的过程，对数据载荷添加头部和尾部
解封装 获取数据载荷的过程，去掉报文中的头部和尾部
网关 提供协议转换、路由选择、数据交换等功能的网络设备
路由器 为报文选择传递路径的网络设备
终端设备 数据通信系统的设备，作为数据的发送者或接收者

DNS

DNS是将域名（网址）解析（或叫翻译）成ip地址的工具

端口

端口的作用

当数据被发送到服务器后，服务器通过查看端口识别其他请求的应用服务，将报文发送给相应的程序模块处理

常见的端口号及相关报文

类型 连接方式 端口号
QQ UDP 8000
DNS UDP 53
HTTP（上网） TCP 80
HTTPS（加密上网） TCP 443
tpkt（远程桌面） TCP 3389
dhcp（动态获取地址） UDP 67&&68
telnet（远程管理和控制） TCP 23
SSH（远程控制） TCP 22
FTP（文件传输） TCP 20&&21

网络协议模型

现在主流的TCP/IP模型是参考OSI模型

OSI模型后三层分的太过于细，现实中并不需要这么细，所以有了现在的TCP/IP模型

TCP/IP各层中常见的协议

华为ENSP

基本指令简介：

< > 用户视图（user-viewer）模式：权限低，只能执行调试、查看类指令

[ ] 系统视图（system-viewer）模式：权限高，能对设备配置进行修改

回车符

more：回车键显示一行，空格键翻一页，其他键中止显示

？ 命令查看符（任何模式都能使用）

Ethernet 接口网速：100Mbps

GigabitEthernet 接口网速：1000Mbps

physical：down 物理状态：表示线路没插好，插好则为 up

protocol：down 协议状态：表示IP地址配置不正确，正确则为 up

ctrl + z 直接回到用户模式

tab 补全命令

用户视图指令

< > system-view 由用户模式进入系统模式[ ]

< > language-mode Chinese 将日志提示语言更改为中文

（不常用）< > clock timezone time-zone-name{ add | minus } offset 用来对本地时区信息进行设置

（不常用）< > clock datetime [utc] HH:MM:SS YYYY-MM-DD 用来设置设备当前或UTC日期和时间

< > save+敲两下回车 保存

< > display current-configuration 查看当前正执行的所有配置

< > display save-configuration 查看保存的配置

< > display ip interface brief 显示接口摘要信息

< > reset save-configuration 清空配置

系统视图指令

[ ] quit 退出当前模式

[ ] sysname xx 将设备名更改为xx

[ ] undo info-center enable 关闭信息中心，不再弹出日志（注：undo xxx 有 关闭xxx，删除xxx，使xxx不生效 的意思）

[ ] int e0/0/0 进入接口 e0/0/0

进入接口后才能进行配置的命令

[ ] ip add 192.168.1.1 24 配置IP地址为192.168.1.1 24为掩码

[ ] shutdown 关闭接口

[ ] undo shutdown 激活接口

配置用户通过password方式登录设备

[ ] user-interface vty 0 4

[ ] set authentication password cipher ************ cipher为display时密码加密显示，还可以写为simple直接显示密码，*为密码

用法：用来进入指定的用户视图并配置用户认证方式为password。系统支持用户界面包括Console用户界面和VTY用户界面，Console界面用于本地登录，VTY界面用于远程登录。默认情况下，设备一般最多支持15个用户同时通过VTY方式访问。

常用的系统快捷键

  • ctrl+a：将光标移动到当前行的开头（有时会是选中全网，感觉不太好用）
  • ctrl+c：中止当前命令运行
  • ctrl+e：将光标移动到当前行的末尾
  • ctrl+z：返回到用户视图模式
